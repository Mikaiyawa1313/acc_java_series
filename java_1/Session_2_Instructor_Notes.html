  4. Classes Exercise:

    Rectangle

    Create a Rectangle class with instance variables (attributes, state) for the length and width, both floating point numbers. The class should have at least two constructors. The default constructor, Rectangle(), should initialize the instance variables to 1.0. The class should have setters and getters; the setters must ensure that the parameters are between 0.0 and 50.0. The class should have methods that return the perimeter and the area of the shape. Create an application that instantiates at least as many objects as constructors in Rectangle and exercises all methods of Rectangle.
